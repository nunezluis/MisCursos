\section{Integrales del movimiento}
\frame{
  \frametitle{Primeras integrales del movimiento}
   \begin{itemize}  
  	\item<1-> Las cantidades conservadas, $I_k\left(q_j, \dot{q}_j\right)=C_k$, donde $C_k=$ constante, y $k=1, \ldots, n$
, son primeras integrales del movimiento de un sistema.
	\item<2-> Un sistema con $s$ grados de libertad es integrable si posee $s$ cantidades conservadas; es decir, si $n=s$.
	\item<3-> A partir de esa integración las demás coordenadas pueden, en principio, ser integradas. 
	\item<4-> Un sistema para el cual existen más cantidades conservadas que grados de libertad $(n>s)$ se llama superintegrable.
	\item<5-> El ejemplo más simple de sistema superintegrable es una partícula libre. Otro ejemplo es el problema de dos cuerpos sujetos a interacción gravitacional.
	\item<6-> Si un sistema con $s$ grados de libertad tiene menos de $s$ cantidades conservadas $(n<s)$, se denomina no integrable.
   \end{itemize}
}

\section{Sistemas Integrables}
\frame{
  \frametitle{Sistemas Integrables}
   \begin{itemize}
   	\item<1-> Ejemplos de sistemas integrables
	\begin{itemize}
		\item Oscilador armónico simple: $s=1 ; C_1=E=$ cte, $n=1$; es integrable.
		\item  Péndulo simple: $s=1 ; C_1=E=\mathrm{cte}, n=1$; es integrable.
		\item  Partícula sobre un cono: $s=2, C_1=l_z=$ cte, $C_2=E=$ cte, $n=2$; es integrable.
		\item  Péndulo doble: $s=2 ; C_1=E=$ cte, $n=1$; no es integrable.
		\item  Péndulo cuyo soporte gira en un círculo en plano vertical con velocidad angular constante: $s=1, n=0$; no es integrable.
		\item  Péndulo de resorte: $s=2 ; C_1=E=$ cte, $n=1$; no es integrable.
		\item  Péndulo paramétrico cuya longitud varía en el tiempo: $s=1, n=0$; no es integrable.
		\item  Partícula libre es superintegrable: $s=3 ; n=4$ : $C_1=E=$ cte, $C_2=p_x=$ cte, $C_2=p_y=\mathrm{cte}, C_2=p_z=\mathrm{cte}$.
	\end{itemize}  
	\item<2-> La integrabilidad es un tipo de simetría presente en varios sistemas dinámicos, y que conduce a una evolución regular (periódica o estacionaria) de las variables del sistema en el tiempo
    \end{itemize}
}

\section{Integrabilidad }
\frame{
  \frametitle{Integrabilidad }
   \begin{itemize}  
	\item<1->  El Lagrangiano de un sistema unidimensional con coordenada $q$ tiene la forma general $L=T\left(\dot{q}^2\right)-V_{e f}(q)=\frac{1}{2} a \dot{q}^2-V_{\mathrm{ef}}(q)$, donde $a$ representa masa, longitud, etc., y $V_{\text {ef }}(q)$ es un potencial efectivo que depende de la coordenada $q$, 
	\item<2->  Como $\frac{\partial L}{\partial t}=0$, la función de energía se conserva
	\item<3->  Hay un grado de libertad y una cantidad conservada; el sistema es integrable.
	\item<4-> Como $E =cte$, se determina $t(q)$ en términos de una integral explícita, $\dot{q} =\frac{{\rm d} q}{{\rm d} t}=\sqrt{\frac{2}{a}\left(E-V_{\mathrm{ef}}(q)\right)} \Rightarrow t(q)  =\int \sqrt{\frac{a}{2}} \frac{{\rm d} q}{\sqrt{E-V_{\mathrm{ef}}(q)}}+\text { cte. }$
	\item<5-> En principio, se puede invertir $t(q)$ para obtener $q(t)$. 
	\item<6-> Para que la solución $q(t)$ sea real, el movimiento puede ocurrir solamente para valores de $q$ tales que $E \geq V_{\mathrm{ef}}(q)$.
	\item<7-> La condición de integrabilidad de sistemas unidimensionales permite calcular el período de movimientos oscilatorios en esos sistemas.
   \end{itemize}
}

\section{Movimiento de una partícula en un potencial, arbitrario $V(x)$}
\subsection{El Lagrangeano, la Energía y los puntos de retorno}
\frame{
  \frametitle{Movimiento de una partícula en un potencial $V(x)$}
   \begin{itemize}  
  	\item<1-> Consideremos un sistema descrito por el Lagrangiano $L=T-V=\frac{1}{2} m \dot{x}^2-V(x)$, con la ecuación de  movimiento $m \ddot{x}=-\frac{{\rm d} V}{{\rm d} x}$
	\item<2-> La energía total constante es $E=\frac{1}{2} m \dot{x}^2+V(x)$ y podemos integrar $\frac{{\rm d} x}{{\rm d} t} = \sqrt{\frac{2}{m}}\sqrt{E-V(x)} \Rightarrow t(x)=\sqrt{\frac{m}{2}} \int \frac{{\rm d} x}{\sqrt{E-V(x)}}$
	\item<3-> Como $\frac{1}{2} m \dot{x}^2=E-V(x) \geq 0$, el movimiento sólo puede ocurrir para $E \geq V(x)$.
	\item<4-> Los puntos de retorno son aquellos para $V(x)=E$. Es decir, $x_1, x_2$ y $x_3$ son puntos de retorno  $V\left(x_1\right)=E, V\left(x_2\right)=E, V\left(x_3\right)=E$
	\begin{figure}[t]
		\includegraphics[width=2.0in]{Figuras/Potencial.png}
   	\end{figure}
   \end{itemize}
}

\subsection{Los puntos de equilibrio, retorno y período}
\frame{
  \frametitle{Los puntos de equilibrio, retorno y período}
   \begin{itemize}  
  	\item<1-> Los puntos de equilibrio $x=x_0$ son aquellos donde la fuerza instantánea se anula: $f\left(x_0\right)=\left.0 \Rightarrow \frac{{\rm d} V}{{\rm d} x}\right|_{x_0}=0$
	\item<2-> Un punto estático: velocidad y aceleración se anulan, $\ddot{x}=0, \dot{x}=0$.
	\item<3->  Un punto de equilibrio $x_0$ es estable si $x=x_0+\eta$, donde $\eta$ es un pequeño desplazamiento, tiende a $x=x_0$ al aumentar el tiempo y corresponde a un mímino del potencial $V(x)$. Es decir $\left.\frac{{\rm d}^2 V}{{\rm d} x^2}\right|_{x_0}>0, \quad x_0$ es un punto de equilibrio estable
	\item<4-> Un punto de equilibrio es inestable si el potencial $V(x)$ presenta un máximo en ese punto. Entonces $\left.\quad \frac{{\rm d}^2 V}{{\rm d} x^2}\right|_{x_0}<0, \quad x_0$ es un punto de equilibrio inestable.
	\item<5-> El período de oscilación entre los puntos de retorno $x_1$ y $x_2$ es dos veces el intervalo de tiempo del movimiento entre esos puntos, $\tau_p(E)=2 \sqrt{\frac{m}{2}} \int_{x_1}^{x_2} \frac{{\rm d} x}{\sqrt{E-V(x)}}$
    \end{itemize}
}

\subsection{Pequeñas oscilaciones en potenciales arbitrarios}
\frame{
  \frametitle{Pequeñas oscilaciones en potenciales arbitrarios}
  \begin{itemize}  
	\item<1-> Para un valor de $x$ cerca de un punto de equilibrio estable $x_0$, el potencial $V(x)$ puede expresarse mediante una expansión de Taylor alrededor de $x=x_0$. Esto es 
	$V(x)=V\left(x_0\right) +\cancelto{0}{\left.\frac{{\rm d} V}{{\rm d} x}\right|_{x_0}}\left(x-x_0\right) +\left.\frac{1}{2} \frac{{\rm d}^2 V}{{\rm d} x^2}\right|_{x_0}\left(x-x_0\right)^2+\cdots$, donde $V\left(x_0\right)$ es un valor constante y el segundo término se anula debido a la condición de equilibrio. 
	\item<2-> Consideremos un pequeño desplazamiento $\eta$ alrededor de $x_0$, entonces  $x=x_0+\eta$ donde $\eta \rightarrow 0\left(\eta / x_0 \ll 1\right)$.
	\item<3-> Despreciando términos en potencias de $\eta$ de orden superior al cuadrático, tendremos 
	$V(x)=\left.\frac{1}{2} \frac{{\rm d}^2 V}{{\rm d} x^2}\right|_{x_0}\left(x-x_0\right)^2=\frac{1}{2} K\left(x-x_0\right)^2$, donde $K \equiv \left.\frac{{\rm d}^2 V}{{\rm d}    x^2}\right|_{x_0}=\text { constante }>0$
	\item<4-> $V(x)$ posee la misma forma funcional del oscilador armónico.
	\item<5-> Su ecuación de movimiento $m \ddot{\eta}  =-K(x-x_0)=-K \eta$, entonces 
	$\Rightarrow \ddot{\eta}+\omega^2 \eta=0 \Rightarrow \omega^2 \equiv \frac{K}{m}=\frac{1}{m} \left.\frac{d^2 V}{d x^2}\right|_{x_0} \Leftrightarrow \quad \omega^2=\left.\frac{1}{a} \frac{\partial^2 V_{\mathrm{ef}}}{\partial q^2}\right|_{q_0}$
    \end{itemize}
}

\section{Recapitulando}
\frame{
\frametitle{Recapitulando}
\begin{itemize}  
	\item<1-> Esta presentación explora los conceptos de {\bf sistemas integrables}, {\bf no integrables}, {\bf superintegrables} y sus implicaciones en la dinámica de sistemas físicos.
	\item<2->   {\bf Sistemas integrables} permiten una evolución predecible con soluciones exactas.
	\begin{itemize}
		\item Oscilador armónico simple (\( s = 1, n = 1 \)).
		\item Péndulo simple (\( s = 1, n = 1 \)).
		\item Partícula sobre un cono (\( s = 2, n = 2 \),  momento angular y energía).
	\end{itemize}
	\item<3-> {\bf Sistemas no integrables} pueden mostrar caos dinámico.
	\begin{itemize}
		\item Péndulo doble (\( s = 2, n = 1 \)).
		\item Péndulo de resorte (\( s = 2, n = 1 \)).
		\item Péndulo paramétrico con longitud variable (\( s = 1, n = 0 \)).
	\end{itemize}
	\item<4-> La {\bf superintegrabilidad} es una propiedad especial de ciertos sistemas con más constantes de movimiento que grados de libertad. La partícula libre (\( s = 3, n = 4 \), energía y 3 momentos lineales).
	\item<5-> Movimiento en un Potencial \( V(x) \):  Puntos de equilibrio \( \left. \frac{dV}{dx}\right|_{x_0} = 0 \). Estables: Si \( \frac{d^2V}{dx^2} > 0 \); Inestables: Si \( \frac{d^2V}{dx^2} < 0 \); Cálculo del período.
    \end{itemize}
}

\section{Para la discusión}
\frame{
\frametitle{Para la discusión}
\begin{itemize}  
	\item<1-> Considere el oscilador armónico para una energía $E$ conocida.
		\begin{itemize}
		\item ¿El sistema es integrable? ¿Por qué? 
		\item Encuentre el punto de equilibrio, los puntos de retorno y el período.
	\end{itemize} 
	\item<2-> Considere el péndulo simple para una energía $E$ conocida y arbitraria. 
	\begin{itemize}
		\item ¿El sistema es integrable? ¿Por qué? Encuentre el punto de equilibrio, los puntos de retorno y el período.
		\item ¿depende el período de la masa? ¿Depende el período de la energía?
	\end{itemize} 
	\item<3-> Considere una partícula de masa $m$ moviéndose sobre la superficie de un cono vertical con ángulo de vértice $\alpha$. 
	\begin{itemize}
		\item ¿El sistema es integrable? ¿Por qué? Encuentre el Lagrangeano y las primeras integrales
		\item Encuentre los puntos de equilibrio, retorno y frecuencia para pequeñas oscilaciones
	\end{itemize}
\end{itemize}	
}
